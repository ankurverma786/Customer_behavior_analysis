# Customer Behavior Analysis Dashboard

## Overview
This is an end-to-end Data Analytics project focused on customer behavior analysis. The project includes data cleaning using Python, SQL-based analysis, and an interactive Power BI dashboard for visualization.

## Tools Used
- Python
- Pandas & NumPy
- SQL
- Power BI

## Project Workflow
- Data Cleaning & Preprocessing using Python
- Exploratory Data Analysis (EDA)
- SQL Analysis
- Dashboard Creation in Power BI

## Dashboard Insights
- Revenue by Category
- Sales Analysis
- Customer Segmentation
- Subscription Status Analysis
- Revenue by Age Group

## Key Features
- Interactive Power BI Dashboard
- KPI Cards & Visual Reports
- Business Insights from Customer Data

## Files Included
- Python preprocessing files
- SQL query files
- Power BI dashboard (.pbix)
- Dataset files
